## What is Zanode?

Zanode is a cloud hosting platform built specifically for South African developers. Deploy your applications with a simple `git push` and enjoy local latency, managed databases, and predictable ZAR billing.

## Supported frameworks

Zanode auto-detects and builds your favourite frameworks:

- **Next.js** — Full-stack React with SSR
- **SvelteKit** — Svelte framework with SSR
- **Astro** — Static site generator
- **Nuxt** — Vue.js framework
- **Remix** — React framework
- **Vite** — React, Vue, and Svelte SPAs
- **Node.js** — Express, Fastify, and other servers
- **WordPress** — One-click managed WordPress hosting
- **Docker** — Bring your own Dockerfile
